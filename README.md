# 1. 前言

## 1.1 原则

1. 每篇笔记必须开头附带前言，进行解释。
2. 前言内必须包含：
   * 学习前提
   * 笔者使用的环境
     * IDE（集成开发环境）
     * SDK（开发工具）
     * OS（操作系统）
     * Plug-in（开发中使用到的插件）
   * 学习资料
     * 前提资料
     * 参考资料（平时解决的问题应该直接在问题后附带上连接）
3. 讲解目的/为什么要讲解（What）
4. 怎样讲解（How）
5. 讲解后有什么作用（Why）
